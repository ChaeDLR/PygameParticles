## Mac OS venv
___

### Create virtual environment files in your current directory

```
python3 -m venv (folder name)
```

### Activate environment

```
source (folder name)/bin/activate
```

## Windows 10 venv
___

### Create virtual environment files in your current directory

```
python -m venv (folder name)
```

### Activate environment

```
(folder name)/Scripts/activate.bat
```

#### To deactivate venv

```
deactivate
```
